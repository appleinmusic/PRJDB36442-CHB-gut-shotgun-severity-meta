# 临床/公共卫生重要性（Clinical Importance）

## 为什么是乙肝？
慢性乙肝患者在长期管理中即便达成病毒学控制，仍可能出现残余炎症、纤维化进展与肝癌风险；这提示单纯抑制病毒并不能完全解释/解决疾病进展。

## 为什么是“肠-肝轴/菌群功能模块”？
肝脏与肠道通过门静脉与胆汁循环紧密相连；肠道微生物可通过代谢物（如次级胆汁酸、色氨酸代谢产物、短链脂肪酸）与结构成分（如 LPS）影响肝脏免疫与炎症-纤维化通路。

## 该课题可能改变什么？
1) 将“病毒学控制后的残余风险”分解为可量化的微生态功能模块；
2) 为风险分层、随访策略与潜在微生态/代谢干预提供可验证的靶点优先级；
3) 提供可复现的数据分析与资源，为后续多中心研究或转化研究奠基。
